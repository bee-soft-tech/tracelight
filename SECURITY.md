# Security Policy

## Reporting a vulnerability

If you discover a security issue in Tracelight, please **do not open a public issue**. Instead,
report it privately via GitHub's **[Security Advisories](https://github.com/bee-soft-tech/tracelight/security/advisories/new)**
(Security → Report a vulnerability).

Please include a description, affected version(s), and reproduction steps. You can expect an
initial response within a few days. Fixes are released as new versions once verified.

## Supported versions

Tracelight is pre-1.0; only the latest released version receives security fixes.
